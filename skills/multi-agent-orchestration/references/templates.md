# Multi-Agent Orchestrator Templates

## Team Lead Kickoff Template
Use this template to initiate the task breakdown and delegation.

```markdown
# 🏁 Mission Kickoff: [Mission Name]
**Mission Objective:** [Restate the final user goal]

## 📋 Task Breakdown
1. **[Sub-task 1]:** [Description] -> **Owner:** [Employee Agent 1]
2. **[Sub-task 2]:** [Description] -> **Owner:** [Employee Agent 2]
...

## ⛓️ Dependencies & Sequence
- [Sub-task 2] depends on [Sub-task 1].
- [Sub-task 1] and [Sub-task 3] can run in parallel.

## ✅ Success Criteria
- [Criteria 1]
- [Criteria 2]
```

## Employee Task Brief Template
Use this template when assigning a specific sub-task to an Employee.

```markdown
# 🛠️ Task Brief: [Sub-task Name]
**Assigned Owner:** [Employee Agent Name]
**Objective:** [Specific deliverable for this sub-task]
**Dependencies:** [What is needed from other agents]
**Constraints:** [Any boundaries or limits]
```

## Employee Progress Update Template
Use this template for reporting intermediate progress or finding.

```markdown
# 📝 Progress Update: [Sub-task Name]
**Status:** [Starting | In-Progress | Review]
**Assumptions:** [What is being assumed]
**Intermediate Findings:** [Key data discovered so far]
**Next Steps:** [Remaining work]
```

## Blocker Escalation Template
Use this template when an Employee encounters a critical issue.

```markdown
# 🚨 Blocker Escalation: [Sub-task Name]
**Problem:** [Clear description of the blocker]
**Impact:** [What other sub-tasks are affected]
**Proposed Solution:** [Optional: How to move forward]
**Question for Team Lead:** [Specific guidance needed]
```

## Employee Completion Handoff Template
Use this template when an Employee completes their assigned task.

```markdown
# ✅ Task Handoff: [Sub-task Name]
**Objective Completed:** [Summary of work done]
**Deliverable:** [Link or content of the deliverable]
**Observations:** [Anything notable for the Team Lead]
**Status:** COMPLETE
```

## Final Synthesis Template
Use this template to consolidate all Employee outputs into a final response.

```markdown
# 🏆 Mission Complete: [Mission Name]
**Final Goal Met:** [How the objective was achieved]

## 📊 Consolidated Result
[Polished, integrated output from all sub-tasks]

## 💡 Key Insights
- [Insight 1]
- [Insight 2]
```

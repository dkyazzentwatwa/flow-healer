# Code of Conduct

## Our Commitment

We want Flow Healer to be a welcoming, respectful, and constructive project for everyone.
Contributors, maintainers, and community members are expected to treat each other with
professionalism, patience, and empathy.

## Expected Behavior

Examples of behavior that helps create a positive environment include:

- Using respectful, inclusive language
- Giving and receiving feedback constructively
- Assuming good intent while still addressing real problems directly
- Focusing on what is best for the project and community
- Showing kindness to people who are learning or asking basic questions

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment, intimidation, or discrimination of any kind
- Personal attacks, insults, or hostile trolling
- Publishing private information without explicit permission
- Sexualized language or imagery in project spaces
- Repeatedly derailing technical discussion or review
- Any other conduct that a reasonable person would consider inappropriate in a professional setting

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, code review,
discussions, and other public or private interactions that are clearly tied to this project.

## Reporting

If you experience or witness unacceptable behavior, contact the project maintainers through a
private maintainer contact channel when possible. If a private channel is not obvious, avoid
posting sensitive details publicly; instead, open a minimal issue asking a maintainer to provide a
private way to continue the conversation.

All reports will be reviewed promptly and handled as respectfully and confidentially as practical.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. They may
remove, edit, or reject comments, commits, code, issues, and other contributions that are not
aligned with this policy, and may temporarily or permanently restrict participation when necessary.
